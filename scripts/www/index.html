计价说明

含钼废钢中的废钢价格，按《天津钢管制造有限公司废钢收购价格表》执行。

含钼废钢计价公式

Pt×（1-Y）+（Pm÷60%×Y×Z） (元/吨)

  • Pt表示废钢价格
  • Y表示钼含量百分比
  • Pm表示钼铁价格
  • Z表示钼铁回收折扣值（当Y<0.2%，Z=70%；0.2%≤Y<0.3%，Z=75%；Y≥0.3%，Z=80%）
  • 60%表示钼标钢钼含量

计价说明

含钼刨花中的废钢价格，按《天津钢管制造有限公司废钢收购价格表》执行。

含钼刨花计价方式

  • 0.15% ≤ Mo < 0.2%：300元/吨 + 当期刨花价格
  • 0.2% ≤ Mo < 0.3%：450元/吨 + 当期刨花价格
  • 0.3% ≤ Mo < 0.4%：650元/吨 + 当期刨花价格
  • Mo ≥ 0.4%：900元/吨 + 当期刨花价格

使用帮助

含钼废钢计算器使用步骤：

  1. 输入废钢价格 (Pt)
  2. 输入钼含量百分比 (Y)
  3. 输入钼铁价格 (Pm)
  4. 回收率系数 (Z) 将根据钼含量自动计算
  5. 输入废钢重量（可选，默认为1吨）
  6. 调整税率（可选，默认12.5%）
  7. 输入费用（可选，默认为0元/吨）
  8. 点击"计算价格"按钮

含钼刨花计算器使用步骤：

  1. 输入当期刨花价格
  2. 输入钼含量 (Mo)
  3. 输入刨花重量（可选，默认为1吨）
  4. 调整税率（可选，默认12%）
  5. 输入费用（可选，默认为0元/吨）
  6. 点击"计算价格"按钮

注意：本计算器仅供参考，实际价格以最终商务谈判为准。
